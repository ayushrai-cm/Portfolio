# Portfolio Website

A personal portfolio website built to showcase my skills, projects, and achievements as a developer. It provides an overview of my work, technical abilities, and ways to connect with me.

##  Features

- Modern and responsive design
- About Me section
- Skills showcase
- Projects display
- Contact section
- Clean UI and smooth navigation

##  How to Run

git clone https://github.com/ayushrai-cm/Portfolio.git
cd Portfolio
Open index.html in your browser

##  Tech Used

- HTML
- CSS
- JavaScript
